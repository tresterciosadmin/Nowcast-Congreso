# Archivos_Borrar

Carpeta de **descarte manual**. Acá va todo lo temporal/regenerable que se produce durante el trabajo: cachés, descargas crudas, logs de validación, salidas intermedias, archivos de prueba.

Por qué existe: la carpeta del proyecto está sincronizada con OneDrive y el entorno **no puede borrar** archivos (solo crearlos). Entonces, en vez de dejar basura mezclada, todo lo descartable se concentra acá y **vos lo borrás a mano** cuando querés.

Regla para cualquier Claude/persona: si un archivo es regenerable o solo sirvió una vez, escribilo (o movelo) acá, no en el módulo. Nada en `Archivos_Borrar/` es fuente de verdad.
